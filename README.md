## рџљЂ Getting Started

### Option 1: Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/unclray/www.unclrayfoods.co.za-.git
   cd www.unclrayfoods.co.za-
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (if installed)
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

### Option 2: GitHub Pages

This repository is configured for GitHub Pages. To enable it:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select `main` branch as source
4. Your site will be available at `https://unclray.github.io/www.unclrayfoods.co.za-/`

## рџЋЁ Customization Guide

### Change Colors
Edit the CSS variables in `styles/style.css`:
```css
:root {
    --primary-color: #2c5f2d;        /* Green */
    --secondary-color: #f39c12;      /* Orange */
    --text-color: #333;
    --light-bg: #f5f5f5;
    --white: #ffffff;
}
```

### Update Company Information
- Update company name in `index.html`, `about.html`, etc.
- Update contact details in `contact.html`
- Add product images to `styles/` folder and update image paths

### Add Products
- Edit `products.html` to add or modify product cards
- Update prices and descriptions
- Add product images
